# 贡献

```
mvn install

## Generate InteliJ IDEA Project
mvn idea:idea       # do your magic

## Logging, set CHATOPERA_SDK_LOG_TRACE as ENV
CHATOPERA_SDK_LOG_TRACE=on

## Generate JavaDocs and Site
./scripts/docs.sh

## CLI assets
cd assets # Read assets/README.md to get usage.
```
